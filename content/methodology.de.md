Das Wörterbuch strebt keine strikte enzyklopädische Neutralität an. Es nimmt vielmehr eine kritische, situierte und überarbeitbare Perspektive ein.

Jeder Eintrag kann verbinden:

- eine Arbeitsdefinition;
- eine Genealogie oder einen Entstehungskontext;
- theoretische oder politische Spannungen;
- Verweise auf benachbarte Begriffe;
- eine kurze Bibliografie.

Da die Website in Git ediert wird, kann Dissens produktiv bleiben: Vorschlag, Diskussion, Revision und Nachvollziehbarkeit gehören zur redaktionellen Methode.
